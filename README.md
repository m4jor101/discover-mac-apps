## 📖 Legend

**Pricing**
- 🆓 **Free**: Completely free to use
- 🔓 **Freemium**: Core features are free, advanced features require payment
- 💳 **One-Time**: Pay once, keep forever
- 🔄 **Subscription**: Recurring payment model

**Architecture**
- ⚡️ **Apple Silicon**: Natively optimized for M-series chips
- 💻 **Intel**: Runs natively on Intel Macs (or via Rosetta on Apple Silicon)
- 🌐 **Universal**: Contains native binaries for both Intel and Apple Silicon
